# 📰 Django Article Website

A dynamic **Article Website** developed using **Python, Django, HTML, and CSS**. The application displays articles across multiple categories and demonstrates the core concepts of the Django framework, including URL routing, views, templates, template inheritance, and context data.

## 🚀 Features

* Home Page
* Blogs Page
* Sports Page
* News Page
* International Page
* About Page
* Read More Functionality
* Responsive Navigation Bar
* Dynamic Content Rendering

## ▶️ How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Install Django:

   ```bash
   pip install django
   ```
4. Run the development server:

   ```bash
   python manage.py runserver
   ```
5. Open your browser and visit:

   ```
   http://127.0.0.1:8000/
   ```
